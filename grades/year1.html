Logic and Proof

Year 1, Semester 1. Grade: A (71%)
71%

This module introduced me to the mathematical concept of logic. This included predicate logic, including the truth tables of AND, OR, IMPLICATION, BI-IMPLICATION and NOT. And First-Order Logic. We also covered areas such as parse trees, Boolean algebra, sets and Russell's Paradox.

Interaction Design

Year 1, Semester 1. Grade: A (71%)
71%

This module covered the creation of Android apps and was entirely coursework based. I built a currency converter and a game called Hit the Target using MIT App Inventor and Java/XML respectively. The course focused on creating apps that were useful and attractive to use and look at.

Praxis

Software Development 1

Year 1, Semester 1. Grade: B (67%)
67%

As the first of 3 Software Development modules this course introduced us to the concept of object-oriented programming and in particular the programming language Java. We covered objects/classes and types, variables, loops, basic data structures and some sorting algorithms.

Introduction to Computer Systems

Year 1, Semester 2. Grade: D (48%)
48%

Intro to Computer Systems covered Computing at a much lower level than I had worked with before. The course started off covering the history of computing but later topics discussed information representation and floating point binary values, processor design, operating systems, memory and virtual memory, and MIPS Assembly language.

Software Development 2

Year 1, Semester 2. Grade: C (59%)
59%

Software Development 2 covered some more complex areas of programming in Java, such as abstract classes, polymorphism, refacturing and inheritance. Part 2 of this module discussed graphical user interfaces (GUIs) in Java (using Java AWT and Swing libraries).

Software Development 3

Year 1, Semester 2. Grade: C (50%)
50%

This module taught us more complex data structures such as Stacks, Queues, Linked Lists and Trees, and algorithms for searching and sorting the data being stored in them. We also covered Big-Oh notation on this course which can be used to measure the efficiency of algorithms and compare them against each other.

Web Design and Databases

Year 1, Semester 2. Grade: A (75%)
75%

In this module we were introduced to creating websites using HTML and CSS. We were taught some theory about how people use websites and the way in which websites are changing (e.g responsive web design more important now). We later started working on databases using MySQL and learned to create forms in websites which could communicate with the database through PHP.
