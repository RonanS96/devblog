Module Library

A small module library that I will write and test myself.



I aim to build a Java module library which can help to reduce time testing and writing code for me in the future. This will include methods for searching and sorting elements within different data structures. The first class to be added to the module library, IntSorter, will include Java methods for sorting arrays of integers in ascending and descending order. Once I have made some progress the source code will be available on GitHub. I have used VisuAlgo.net to reference the pseudocode for most of the methods in this class.

There will also be code fragments included for simple but commonly used actions such as finding the maximum and minimum values in a list.

Some other classes to be added include Stack and Queue ADTs and searching algorithms.
