Alexa Skill: Hearts Facts

An Amazon Alexa skill which tells you interesting facts about Heart Of Midlothian F.C.

‚Üê Back to personal projects



After seeing an advertisment for Alexa skills development on Facebook I thought I would give it a go. First, I decided to do a tutorial provided on this page. I selected the Fact Skill tutorial and using the information given in the tutorial and the template code I was able to create this skill.

The skill I built is based around Heart of Midlothian Football Club and contains around 40 different facts about the clubs history and honours as well as the great rivalry of the Edinburgh derby. There were 6 steps to the tutorial. First was getting things set up in the Amazon developer portal and creating a new Intent. The intent had to be provided with key phrases which would prompt Alexa to reply, for example "Alexa, tell me a hearts fact". The next stage was to create a lambda function which stored the suitable replies. Step three involved connecting the two, and steps 4 and 5 were testing and customising the skill. Stage 6 was applying for the skill to be certified and published, which I am now waiting to hear back from. I hope that this can be the first step of many in developing Amazon Alexa Skills.

The source code for the intent (or Voice-User Interface) and the lambda function are available on Github.
