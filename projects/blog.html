My Development Blog

A blog which aims to describe me and my work in a simple but interesting way.


I have created a simple 'development blog' to show off some of my work as well as to post updates on my path through university. In addition I will be writing some small articles on computer science related subjects, the first of which is Formal Languages, which talks about what is meant by the term 'formal language', what they are used for and how they are defined. If you are reading this, then my blog is working (at least partly).

To get started, I am using static HTML pages where I can update blog items and information on my university and personal projects simply through editing the source code. Each page should follow the same style as the others and each page will contain a navigation bar at the top and bottom so you can move between pages on the site with ease. The consistent style of the webpages is controlled through a main CSS file.

Over time I will look to improve the site so that the blog can be updated via some user interface and will store information in a database.

I have decided to code from scratch, i.e not using any templates or frameworks, but this may change in a future iteration of the development of this blog.
