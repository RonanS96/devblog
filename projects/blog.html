My Development Blog

A blog which aims to describe me and my work in a simple but interesting way.


This page describes my project that I am undertaking to create a simple development blog where I can constantly provide information about myself in a fairly simple manner. If you are reading this, then my blog is working (at least partly).

To get started, I aim to use very simple static HTML pages where I can update blog items and information on my university and personal projects simply through editing the source code. Each page should follow the same style as the others and each page will contain a navigation bar at the top and bottom so you can move between pages on the site with ease.

Over time I will look to improve the site so that the blog can be updated via some user interface and will store information in a database.

I have decided to code from scratch, not using any templates such as Jekyll or WordPress for example, but this may change in a future iteration of the development of this blog.
