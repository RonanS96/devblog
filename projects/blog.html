My Development Blog

A blog which aims to describe me and my work in a simple but interesting way.

← Back to personal projects



I have created a simple 'development blog' to show off some of my work as well as to post updates on my path through university and work. If you are reading this, then my blog is working (at least partly).

It has been built using HTML, CSS and JavaScript, and currently runs as a static website with no database connection. Originally these were just working alone but now the Bootstrap and W3.CSS frameworks are helping out with style as well. In terms of style I have went with a dark theme, however I have added more colour over time. Style consistency and responsiveness to screen sizes are also important to me and so I have tried to apply the relevant features as much as possible.

The site is being hosted using Amazon Web Services (AWS), with a secure connection being implemented through Amazon CloudFront.
