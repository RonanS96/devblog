Amy's Portfolio

An online architectural technologist's portfolio.

← Back to personal projects



This project involved creating a small website for Amy Dickson to show off some of her impressive work as an architectural technology student at Edinburgh Napier University. She had already created a PDF portfolio which summarised the work she had done over the last year but felt that she would like to go one step further and have an online presence as well. The online portfolio allows for anyone to access it meaning Amy could put the website URL on her CV and possible employers could have a look at it to find out more about her and her work. As well as information on Amy's projects, the website also includes an 'about me' section and a contact section - which uses a Google maps API to show the location of her university. Although the creation of the site is completed, I will be maintaining it and adding new content as it becomes available.

The website has been created using the W3.CSS 'Parallax' template which provides funky image effects when you scroll down the page. It has been created using the 'big 3' in web development - HTML, CSS and a little bit of JavaScript.

URL: https://amy-dickson.com
